# ThesisMEMO
Репозиторий содержит инструкции по прохождению квеста по подготовке к защите ВКР бакалавров и магистров ИВМиИТ КФУ.
Инструкции формировались в 2026ом году. В последующие годы все может поменяться, поэтому нужно каждый год актуализировать сведения.

# Файлы репозитория
1. CheckList.md -- список пунктов, которые необходимо выполнить, чтобы прийти к моменту сдачи готовых документов на кафедре. Фактически этот файл нужно распечатать и просто ставить галочки по мере выполнения этих заданий.
2. Instructions.md -- свалка сообщений, которые поступали студентам из разных (официальных) источников в июне 2026 года. Этот файл является скорее справочным и должен порождать вопросы, которые нужно уточнить к моменту защиты после 2026 года.
3. Memo.md -- попытка структурировать сведения в единую сводку правил.
4. Памятка руководителя ВКР.docx -- этот файл рассылалс вначале 2026 года и содержит требования к защите, которые необходимо иметь ввиду за полгода до конца обучения.
5. Docs -- папка с примерами и шаблонами документов, которые необходимо подготовить к защите.
6. AI Skills -- @rokokol задонатил SKILL для Cluade Code, чтобы писать по ГОСТу. Инструкция в папке.
