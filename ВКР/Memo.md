
0. Даты нигде в документах не проставляются! На кафедре скажут, какую дату ставить.
0.1. Все подписи ставить синей ручкой. Черный цвет не принимается.

1. Для **магистерских диссертаций** рецензент должен быть внешним. то есть его основное место работы не должно совпадать с местом защиты диплома -- КФУ. Для **ВКР** бакалавров допускается рецензент из той же организации, может быть с другого института в КФУ.
2. Рецензент должен быть со степенью (к.н., д.н., либо PhD). Допускаются **физ.-мат.** и **технические** науки. Тематика научной работы рецензента не важна.
3. Для **внешнего рецензента** подпись должна быть заверена в отделе кадров по его месту работы. См. пример в файлах проекта.
4. Защиты ВКР обычно назначаются в период 20--25 июня. Все документы должны быть оформлены и согласованыы на кафедре **за 7 дней до защиты**. То есть к 10--15 июня. Рецензенту нужно время, чтобы подготовить рецензию, подписать ее, заверить в отделе кадров по своему месту работы и (возможно) отправить в Казань. Поэтому диплом хотя бы в приличном черновом варианте должен быть готов к 1 июня вместе с проектом рецензии для передачи рецензенту.
5. **Диплом** оформляется по ГОСТу. Есть ГОСТ для офомления текста диплома и отдельный ГОСТ регламентирует оформление **Списка использованных источников**.
6. **Диплом** должен включать **Титульный лист**, **ВВЕДЕНИЕ**, **Основной текст**, **ЗАКЛЮЧЕНИЕ**, **СПИСОК ИСПОЛЬЗОВАННЫХ ИСТОЧНИКОВ**. Нумеруются только разделы, подрзделы, пункты, параграфы и проч основного текста.
7. В копицентре будут предлагать прошить вместе с листами диплома еще и файлы, в которые можно будет вложить дополнитеьные документы-приложения. По состоянию на **10.06.2026** такие файлы не нужны. Прошивается/сброшуровывается только титульный лист и текст диплома.
8а. Для успешной защиты **ВКР бакалавров** требуется: (i) доклад на студенческой конференции **в апреле**, (ii) публикация тезисов доклада студенческой конференции, (iii) публикация по теме дипломной работы научной статьи уровня РИНЦ (или выше) либо регистраия программы для ЭВМ, (iv) полностью готовая ВКР.
8б. Для успешной защиты **ВКР магистров** требуется: (i) доклад на студенческой конференции **в апреле**, (ii) публикация тезисов доклада студенческой конференции, (iii) публикация по теме дипломной работы научной статьи уровня ВАК/ядро РИНЦ/Scopus/WoS (регистрация программы для ЭВМ -- недостаточно), (iv) полностью готовая ВКР. Хороший доклад могут зачесть во время ГЭК.
